# Valery's Coffee

Интернет-магазин премиального кофе с каталогом, корзиной, заказами в Telegram и защищенной админкой.

## Демо

- Сайт: https://coffee-catalog-telegram-orders.vercel.app/
- Админка: https://coffee-catalog-telegram-orders.vercel.app/admin
- GitHub: https://github.com/it706/coffee-catalog-telegram-orders

## Задача проекта

Создать MVP интернет-магазина для кофейного бренда Valery's Coffee: клиент выбирает товары, собирает корзину и отправляет заказ, а владелец получает заявку в Telegram и может управлять каталогом через админку.

## Что реализовано

- Премиальный адаптивный дизайн для кофейного бренда.
- Каталог зерна, дрип-пакетов и аксессуаров.
- Фильтрация товаров по категориям.
- Карточки товаров с фото, ценами и характеристиками.
- Корзина с изменением количества товаров.
- Анимация добавления товара в корзину.
- Форма заказа с маской телефона.
- Выбор способа получения: самовывоз, доставка по городу, доставка СДЭК.
- Адрес самовывоза: Москва, Лермонтова 51.
- Отправка заказа администратору в Telegram.
- Генерация номера заказа.
- Защищенная админка по паролю.
- Редактирование товаров, цен, описаний и фото.
- Публикация и скрытие товаров без удаления.
- Деплой на Vercel.

## Как работает заказ

1. Клиент выбирает товары в каталоге.
2. Добавляет товары в корзину и меняет количество.
3. Заполняет имя, телефон и способ получения.
4. Сайт отправляет заказ через API route `/api/order`.
5. Владелец получает уведомление о заказе в Telegram.

## Админка

Админка доступна по адресу `/admin`.

Возможности:

- вход по паролю;
- редактирование названия товара;
- изменение цены;
- изменение категории;
- изменение описания и характеристик;
- выбор фото товара;
- скрытие и публикация товара;
- удаление товара;
- добавление нового товара;
- сброс каталога к стартовой версии.

Важно: текущая админка является demo/MVP-версией и хранит изменения в `localStorage` браузера. Для реального коммерческого запуска следующим шагом нужно подключить базу данных и полноценную авторизацию.

## Переменные окружения

Создайте файл `.env.local`:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_CHAT_ID=ваш_chat_id
ADMIN_PASSWORD=пароль_для_админки
```

Файл `.env.local` нельзя выкладывать на GitHub.

## Локальный запуск

```bash
npm install
npm run dev
```

Если PowerShell блокирует `npm`, используйте:

```powershell
npm.cmd run dev
```

Сайт будет доступен по адресу:

```text
http://localhost:3000
```

Админка:

```text
http://localhost:3000/admin
```

## Результат

Проект показывает полный пользовательский сценарий малого бизнеса: витрина, корзина, оформление заказа, уведомление владельца в Telegram и базовое управление каталогом. Подходит для демонстрации навыков создания коммерческих сайтов и MVP-сервисов через Next.js и AI-инструменты.
