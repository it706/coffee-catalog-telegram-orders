# Valery's Coffee - интернет-магазин кофе с заказами в Telegram

Учебно-коммерческий интернет-магазин премиального кофе и аксессуаров на Next.js.

## Задача

Создать интернет-магазин с фильтрами, корзиной и отправкой заказа администратору в Telegram.

## Что реализовано

- Каталог кофе и аксессуаров.
- Фильтр по категориям.
- Корзина с количеством товаров и итоговой суммой.
- Форма клиента с маской телефона.
- Демо-админка каталога на странице `/admin` с входом по паролю.
- API endpoint `/api/order`.
- Отправка заказа в Telegram Bot API через переменные окружения.

## Технологии

- Next.js
- React
- TypeScript
- CSS
- Telegram Bot API

## Переменные окружения

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_CHAT_ID=ваш_chat_id
ADMIN_PASSWORD=пароль_для_админки
```

Файл `.env.local` нельзя выкладывать на GitHub.
